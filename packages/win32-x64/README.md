# @native-audio-node/win32-x64

This package contains the pre-built native binary for `native-audio-node` on Windows x64.

**This package is automatically installed** as an optional dependency when you install `native-audio-node` on a compatible platform.

## Do not install directly

```bash
# Do this instead:
npm install native-audio-node
```

## Platform Support

- **OS:** Windows (win32)
- **Architecture:** x64
- **Windows Version:** Windows 10 2004+ (Build 19041 or later)
